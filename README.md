# P3L Backend

Node.js Express backend for P3L system. Uses MySQL (XAMPP) for database. Provides REST API endpoints for all frontend features (assignments, labs, classes, fees, messaging, GitHub integration, authentication).

## Structure
- src/routes: Express route definitions
- src/controllers: Route handlers
- src/models: Database models
- src/config: Configuration (DB, env)

## Setup
1. Install dependencies: `npm install`
2. Configure `.env` for MySQL connection
3. Start server: `npm run dev`
